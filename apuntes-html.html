Apuntes de HTML - Repaso para el Examen

1. Estructura básica de HTML

<!DOCTYPE html>
<html>
  <head>
    <title>Mi Página</title>
  </head>
  <body>
    <!-- Contenido aquí -->
  </body>
</html>

2. Títulos y párrafos

<h1> hasta <h6>: Títulos (de más grande a más pequeño)

<p>: Párrafo de texto

3. Negrita, cursiva y subrayado

<strong>: Negrita

<em>: Cursiva

<u>: Subrayado

4. Listas ordenadas y desordenadas

<ol>: Lista ordenada

<ul>: Lista desordenada

<li>: Elemento de lista

5. Imágenes

<img src="url" alt="Texto alternativo">

src: indica la URL de la imagen

alt: texto alternativo si no carga

6. Enlaces

<a href="url">Texto del enlace</a>

href: indica a dónde lleva el enlace

7. Botones

<button>Texto del botón</button>

8. Saltos de línea y líneas horizontales

<br>: Salto de línea

<hr>: Línea horizontal

9. Consejos importantes

  • Las etiquetas van entre < y >
  • Muchas etiquetas tienen cierre: </etiqueta>
  • Siempre escribe HTML dentro de <html> y <body>

10. Recomendaciones para el examen

  • Leer bien cada pregunta
  • Recordar qué hace cada etiqueta
  • Revisar los ejemplos de estos apuntes
  • ¡Practicar es clave!
